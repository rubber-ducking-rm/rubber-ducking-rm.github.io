f:T15e2,<p><a href="https://godot-rust.github.io/book/index.html" rel="nofollow" target="_blank">公式</a>の方が詳しくて勉強になります。
とりあえず動かすだけの手順を抜粋しています。
　</p>
<h1>前提</h1>
<ul>
<li>windows11</li>
<li>Godot:4.2.2</li>
<li>cargo:1.79.0</li>
<li>windows上で正常に動いているプロジェクトに適応</li>
</ul>
<h1>手順</h1>
<h2>emsdkインストール</h2>
<pre><code class="code-highlight"><span class="code-line">git clone https://github.com/emscripten-core/emsdk.git
</span><span class="code-line">cd emsdk
</span><span class="code-line">./emsdk install 3.1.39
</span><span class="code-line">./emsdk activate 3.1.39
</span><span class="code-line">
</span></code></pre>
<h2>rustプロジェクトの編集</h2>
<pre><code class="code-highlight"><span class="code-line">rustup toolchain install nightly
</span><span class="code-line">rustup component add rust-src --toolchain nightly
</span><span class="code-line">rustup target add wasm32-unknown-emscripten --toolchain nightly
</span></code></pre>
<h2>Cargo.toml編集</h2>
<pre class="language-toml"><code class="hljs language-toml code-highlight"><span class="hljs-section">[package]</span>
<span class="hljs-attr">name</span> = <span class="hljs-string">"rust"</span>
<span class="hljs-attr">version</span> = <span class="hljs-string">"0.1.0"</span>
<span class="hljs-attr">edition</span> = <span class="hljs-string">"2021"</span>


<span class="hljs-section">[dependencies]</span>
<span class="hljs-attr">godot</span> = { git = <span class="hljs-string">"https://github.com/godot-rust/gdext"</span>, branch = <span class="hljs-string">"master"</span>, features = [<span class="hljs-string">"experimental-wasm"</span>, <span class="hljs-string">"lazy-function-tables"</span>] }
</code></pre>
<h3>ビルド実行</h3>
<pre><code class="code-highlight"><span class="code-line">cargo +nightly build -Zbuild-std --target wasm32-unknown-emscripten --release
</span></code></pre>
<h2>godotプロジェクトの編集</h2>
<h3>.gdextensionを編集</h3>
<pre><code class="code-highlight"><span class="code-line">[configuration]
</span><span class="code-line">entry_symbol = "gdext_rust_init"
</span><span class="code-line">compatibility_minimum = 4.2
</span><span class="code-line">reloadable = true
</span><span class="code-line">
</span><span class="code-line">[libraries]
</span><span class="code-line">linux.debug.x86_64 =     "res://../rust/target/debug/lib{YourCrate}.so"
</span><span class="code-line">linux.release.x86_64 =   "res://../rust/target/release/lib{YourCrate}.so"
</span><span class="code-line">windows.debug.x86_64 =   "res://../rust/target/debug/{YourCrate}.dll"
</span><span class="code-line">windows.release.x86_64 = "res://../rust/target/release/{YourCrate}.dll"
</span><span class="code-line">macos.debug =            "res://../rust/target/debug/lib{YourCrate}.dylib"
</span><span class="code-line">macos.release =          "res://../rust/target/release/lib{YourCrate}.dylib"
</span><span class="code-line">macos.debug.arm64 =      "res://../rust/target/debug/lib{YourCrate}.dylib"
</span><span class="code-line">macos.release.arm64 =    "res://../rust/target/release/lib{YourCrate}.dylib"
</span><span class="code-line">web.debug.wasm32 = "res://../rust/target/wasm32-unknown-emscripten/release/{YourCrate}.wasm"
</span><span class="code-line">web.release.wasm32 = "res://../rust/target/wasm32-unknown-emscripten/release/{YourCrate}.wasm"
</span></code></pre>
<p>※web.debug.wasm32にdebugをしている指定するとエラーになる</p>
<h3>godotエディタで編集</h3>
<ol>
<li>[プロジェクト]⇒[エクスポート]⇒[追加]⇒[Web]</li>
<li>エクスポート先のパスを任意のパスに設定</li>
<li>ExtensionsSupportを有効化</li>
<li>すべてエクスポートからリリースを選択</li>
</ol>
<h2>動作確認</h2>
<h3>IIS（Internet Information Services）有効化</h3>
<ol>
<li>[windowsキー]⇒[Windowsの機能の有効化または無効化]</li>
<li>[インターネットインフォメーションサービス]をチェック
<img src="/images/godot/IIS.png" alt="インターネットインフォメーションサービス"></li>
</ol>
<h3>IIS マネージャー編集</h3>
<ol>
<li>[windowsキー]⇒[インターネット インフォメーション サービス (IIS) マネージャー]</li>
<li>左メニューから[Default Web Site]を選択</li>
<li>[MIMEの種類]をダブルクリック</li>
<li>右メニューから[追加...] を選択</li>
<li>ファイル名の拡張子を「.pck」、MIMEの種類を「application/octet-stream」と入力
<img src="/images/godot/0713/mime.png" alt="MIME"></li>
<li>左メニューから[Default Web Site]を選択</li>
<li>HTTP応答ヘッダーをダブルクリック</li>
<li>右メニューから[追加...] を選択</li>
<li>以下2つの値をセットする
<pre><code class="code-highlight"><span class="code-line">名前：Cross-Origin-Embedder-Policy
</span><span class="code-line">値：require-corp
</span></code></pre>
<pre><code class="code-highlight"><span class="code-line">名前：Cross-Origin-Opener-Policy
</span><span class="code-line">値：same-origin
</span></code></pre>
</li>
</ol>
